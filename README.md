## 🚀 Как запустить (3 способа)

### Способ 1. Просто открыть файл
1. Распакуйте архив
2. Дважды кликните по файлу `znakvsem-ru.html`
3. Он откроется в браузере — всё работает!

### Способ 2. Залить на хостинг
1. Распакуйте архив
2. Загрузите `znakvsem-ru.html` на любой хостинг в корень сайта
3. Переименуйте в `index.html` (чтобы открывался по `znakvsem.ru/`)
4. Готово — сайт доступен по адресу

### Способ 3. GitHub Pages (бесплатно)
1. Создайте репозиторий на GitHub
2. Загрузите туда `znakvsem-ru.html` (переименованный в `index.html`)
3. Settings → Pages → Source: main branch → Save
4. Через 5 минут сайт будет доступен по `https://USERNAME.github.io/REPO-NAME/`

## 🆘 Частые вопросы

**Q: Что если на сайте Игоря уже есть свой контент?**
A: Этот лендинг можно залить в подпапку (например, `znakvsem.ru/search/`) и ссылаться на неё.

**Q: Можно ли изменить цветовую гамму?**
A: Да. В начале файла есть `<style>` блок с переменными:
```css
:root {
  --bg: #0a0e27;       /* фон */
  --primary: #3b82f6;  /* синий */
  --accent: #06b6d4;   /* голубой акцент */
}
```

**Q: Можно ли добавить Яндекс.Метрику?**
A: Да. Вставьте код счётчика перед `</body>` в конце файла.

**Q: Можно ли убрать ссылку на NayTea?**
A: Можно, но **не рекомендуется** — поисковая форма работает через наш API. Без ссылки пользователи не поймут, куда их перенаправило.

**Q: Что если NayTea упадёт?**
A: Форма не сломается — пользователь увидит ошибку от NayTea. Сам лендинг Игоря продолжит работать. Контакты, услуги, цены — всё доступно.
